# 咨询话术模拟训练系统
## 员工使用说明（1页版）

### 一、文档说明

本说明用于规范日常使用方式，帮助咨询员快速完成查询、学习与训练。

### 二、系统用途

本系统主要用于以下工作：

- 查询患者问题的标准参考回复
- 熟悉常见患者问题与回复规范
- 学习知识库中的优质咨询案例
- 开展对话模拟训练与复盘

### 三、适用场景

1. 新入职咨询员快速熟悉业务内容
2. 接待过程中对某类问题回复不确定时
3. 需要统一回复口径与表达方式时
4. 日常查阅知识条目、巩固业务知识时
5. 通过模拟训练提升接待能力时

### 四、主要功能入口

| 功能模块 | 入口路径 | 功能说明 |
|----------|----------|----------|
| AI 问答助手 | 登录后首页 → `AI 问答助手` | 输入患者问题，获取建议回复；右侧可预览知识库常见问答 |
| 模拟训练 | 顶部菜单 → `模拟训练` | 选取对话案例、开展模拟接待练习 |
| 历史记录 | 顶部菜单 → `历史记录` | 查看已完成的训练记录与系统点评 |

> 说明：知识条目预览（即知识库中已整理的常见 Q&A）在 `AI 问答助手` 页面右侧列显示，无需单独进入菜单。

### 五、标准使用流程

建议按以下流程使用：

1. 登录系统，进入 `AI 问答助手`
2. 在搜索框输入患者原始问题
3. 查看系统建议回复，选择合适内容
4. 结合当前接待情况进行必要调整后使用
5. 如需深入了解，查看右侧知识条目详情
6. 如需系统训练，进入 `模拟训练` 模块

### 六、使用效果

- 减少对常见问题回复时的组织语言负担
- 提高回复的稳定性与一致性
- 加快对标准话术的掌握速度
- 降低新入职咨询员业务上手周期

### 七、使用原则

> 系统建议回复为参考内容，须结合实际接待情况调整后使用。

- 先参考系统建议，再形成最终回复
- 不得照搬内容，应理解后结合语境调整
- 遇到医疗诊断类、敏感类问题，不得依赖系统作答，应转交专业人员

### 八、适合查询的内容

| 问题类型 | 示例 |
|----------|------|
| 地址与来院路线 | 地址在哪里、怎么来 |
| 门诊与接诊时间 | 什么时候可以就诊 |
| 预约与检查流程 | 如何预约、需要什么材料 |
| 收费与医保问题 | 费用多少、能否报销 |
| 常见项目说明 | 某项目是什么、有什么注意事项 |
| 初步安抚与接待 | 患者情绪安抚、等待时间说明 |

### 九、日常使用要求

- 遇到不确定的问题，优先在系统中查询，不得凭主观随意作答
- 查看建议回复后，应先理解其含义再调整使用
- 日常保持浏览知识条目、定期进行模拟训练的习惯
- 发现知识库内容有误或缺漏，应及时向管理员反馈

### 十、结语

本系统服务于日常接待辅助与业务能力提升，是咨询员标准化工作流程的组成部分，请规范使用。
