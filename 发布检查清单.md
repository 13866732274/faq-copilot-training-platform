# ChatTrainer 发布检查清单（5分钟版）

> 适用环境：`/www/wwwroot/chattrainer`  
> 使用方式：上线前逐项勾选，发现异常先停止发布。  
> 更新时间：2026-03-13

---

## 0. 基础可用性（30秒）

- [ ] 后端健康检查通过：`curl -s http://127.0.0.1:8000/api/v1/health`
- [ ] 前端可访问：`http://192.168.9.139`（经 Nginx）
- [ ] 管理员账号可登录（非演示账号）

---

## 1. 导入验证（1分钟）

- [ ] 上传 `.html` 文件可成功预览（消息条数 > 0）
- [ ] 角色识别正确（患者/咨询师方向无明显反转）
- [ ] 导入页"聊天类型"已选择（主动/被动）
- [ ] 点击确认导入成功，题库列表可见新题
- [ ] 题库列表按"聊天类型"筛选结果正确

异常处理建议：

- 若导入失败，先看后端日志 + 文件编码（UTF-8/GBK）；
- 若角色错位，检查 `html_parser` 规则与导出样本结构。

---

## 2. 训练流程验证（1分钟）

- [ ] 学员端能进入"开始练习"列表
- [ ] 聊天页左右分侧正确（患者左 / 学员右）
- [ ] 学员每轮回复 **1次** 后，能推进到下一批患者消息
- [ ] 训练结束出现明确"已结束"提示
- [ ] 可点击"完成练习，查看答案"

---

## 3. 结果与回看验证（1分钟）

- [ ] 训练结果页显示"答题区 / 答案区"左右对比
- [ ] "查看完整原始对话"可展开/收起
- [ ] 历史记录详情页与训练结果页交互一致
- [ ] 管理员批注可新增/编辑/删除（如当前版本启用）

---

## 4. 权限验证（1分钟）

- [ ] 学员账号无法访问管理员页面（如 `/admin/users`）
- [ ] 管理员账号可访问题库、统计、用户管理
- [ ] 未登录访问受保护路由会跳转登录
- [ ] Token 过期后能提示并回到登录流程
- [ ] 权限体检页（超管）：`/admin/permission-audit` 可正常加载

---

## 5. 统计验证（30秒）

- [ ] 首页统计不为异常占位（如 `--`）
- [ ] 统计页"学员维度/题目维度"数据可加载
- [ ] 新增一条练习后，统计能看到变化（可有短延迟）

---

## 6. 备份与回滚准备（1分钟）

- [ ] 已完成数据库备份（至少当天一次）
- [ ] 记录当前后端提交版本/代码快照
- [ ] 保留最近可用发布包（前端 `dist` / 后端代码）
- [ ] 明确回滚命令与负责人（谁执行、何时执行）

建议最小备份项：

- MariaDB：`chattrainer` 库全量导出
- 后端：`backend/` 全目录
- 前端：`frontend/` + 当前构建产物

---

## 7. 发布后3分钟观察项（可选但建议）

- [ ] 后端错误日志无持续 5xx
- [ ] 前端控制台无新增严重报错
- [ ] 导入/开始训练/查看结果至少走通1次
