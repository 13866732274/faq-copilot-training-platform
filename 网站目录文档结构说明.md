## 4. 关键文件映射（排障高频）

| 场景 | 关键文件 |
|------|----------|
| 训练推进逻辑 | `backend/app/api/practice.py` |
| HTML 角色解析 | `backend/app/services/html_parser.py` |
| 题库导入与元数据 | `backend/app/api/quizzes.py` |
| 聊天界面（微信风格） | `frontend/src/views/practice/PracticeChatView.vue` |
| 训练结果统一对比 | `frontend/src/components/practice/PracticeComparePanel.vue` |
| 菜单定义与权限映射 | `frontend/src/constants/menus.ts` |
| RBAC 权限点 | `backend/app/services/permission_points.py` |
| FAQ 处理管线 | `backend/app/services/faq_pipeline.py` |
| FAQ LLM/向量调用 | `backend/app/services/faq_llm.py` |
| FAQ 全部 API | `backend/app/api/faq.py` |
| 系统设置（功能开关） | `backend/app/models/system_setting.py` |
| 路由注册 | `backend/app/main.py`（后端）、`frontend/src/router/index.ts`（前端） |

## 5. 文档维护建议

- 每次新增后端表字段时，同时更新：
  - `alembic/versions/*`
  - `技术实现详细记录.md`
  - 本文档的目录说明（如新增模块）
- 每次新增前端页面时，补充：
  - `router/index.ts` 路由说明
  - `constants/menus.ts` 菜单注册
  - 本文档 `frontend` 结构树
- FAQ、权限审计等新模块如有迭代，同步更新 `技术实现详细记录.md` 对应章节
- **SaaS 模块化改造**的规划详见主项目文档 `12.23` 节
