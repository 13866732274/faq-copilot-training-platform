## 1. 后端技术栈（实际使用）

- 运行框架：`FastAPI`
- 运行服务：`Uvicorn`
- 数据库：`MariaDB 10.11.x`（服务器当前使用 `127.0.0.1:3306`）
- ORM：`SQLAlchemy 2.x`
- 迁移：`Alembic`
- 异步驱动：`asyncmy`
- Alembic 同步驱动：`pymysql`
- 认证：`python-jose`（JWT）+ `passlib[bcrypt]`
- 密码兼容：`bcrypt<4`（兼容 passlib）
- HTML 解析：`BeautifulSoup4`
- 上传处理：`python-multipart`
- HTTP 客户端：`httpx`（调用 Dashscope API）
- AI/LLM：`Dashscope qwen3-max`（问答提取、聚类精炼、Copilot 回复生成）
- 文本向量化：`Dashscope text-embedding-v3`（1024 维度）
- 向量聚类：`numpy` + `scikit-learn`（AgglomerativeClustering）

后端依赖基线见：`backend/requirements.txt`

## 3. 已实现核心能力（按模块）

### 3.1 认证与权限

- 登录：`/api/v1/auth/login`（支持租户编码隔离登录）
- 当前用户：`/api/v1/auth/me`
- 角色：`super_admin / admin / student`
- 权限点系统：`/api/v1/auth/permission-points`
- 前端路由守卫 + 菜单可见性 + 权限点三重控制
- 菜单权限：可配置菜单（admin 级别可裁剪）
- 租户代入模式：超管可代入租户视角操作

### 3.2 题库管理

- 上传预览：`POST /api/v1/quizzes/upload`
- 确认入库：`POST /api/v1/quizzes/confirm`
- 列表/详情：`GET /api/v1/quizzes`、`GET /api/v1/quizzes/{id}`
- 元数据编辑：`PUT /api/v1/quizzes/{id}`（支持标题、聊天类型、分类、难度、标签、描述、患者名、咨询师名）
- 分类标签中心：独立页面管理分类/标签的重命名、合并、删除
- 批量元数据操作：批量设分类、批量加/删标签
- 新增字段：`chat_type`（`active`/`passive`，导入时手动选择）
- 列表筛选：`chat_type`、关键字、分类、标签

### 3.3 训练流程

- 可练习题：`GET /api/v1/practice/available`
- 开始：`POST /api/v1/practice/start`
- 下一批患者消息：`GET /api/v1/practice/{id}/next`
- 回复：`POST /api/v1/practice/{id}/reply`
- 完成：`POST /api/v1/practice/{id}/complete`
- 对比：`GET /api/v1/practice/{id}/review`
- AI 评分：`GET /api/v1/practice/{id}/ai-score`（规则 V1 + LLM V2 融合评分）

训练推进规则（当前版本）：

- 学员每轮回复一次即可推进；
- 系统自动跳过连续咨询师消息；
- 直接推送下一批患者消息（不再要求学员按原始咨询师条数连发）。

### 3.4 记录与统计

- 我的记录：`GET /api/v1/records/my`
- 记录详情：`GET /api/v1/records/my/{practice_id}`
- 统计总览：`GET /api/v1/stats/overview`
- 学员/题目维度统计：`/api/v1/stats/students`、`/api/v1/stats/quizzes`
- 批注增删改：`POST/PUT/DELETE /api/v1/stats/comments/*`

### 3.5 用户管理

- 用户 CRUD、启停、角色筛选、重置密码快捷操作
- 批量导入、批量状态更新、批量菜单权限设置
- 医院/科室绑定（多对多）

### 3.6 FAQ 智能知识库（2026-03-13 新增）

核心流程：`导入对话 → LLM 提取问答对 → 向量化 → 聚类 → LLM 精炼 → FAQ 知识库`

**后端接口：**

| 端点 | 说明 |
|------|------|
| `POST /api/v1/faq/process` | 启动 FAQ 生成管线（后台异步） |
| `GET /api/v1/faq/tasks` | 处理任务列表 |
| `GET /api/v1/faq/tasks/{id}` | 任务详情 |
| `GET /api/v1/faq/clusters` | FAQ 条目列表（分类筛选+关键词搜索+分页） |
| `GET /api/v1/faq/clusters/{id}` | 条目详情（全部问题+答案） |
| `PUT /api/v1/faq/clusters/{id}` | 编辑条目（标题/分类/最佳答案等） |
| `DELETE /api/v1/faq/clusters/{id}` | 删除条目 |
| `POST /api/v1/faq/clusters/merge` | 合并多个条目 |
| `POST /api/v1/faq/answers/{id}/best` | 切换最佳答案标记 |
| `POST /api/v1/faq/answers/{id}/upvote` | 点赞答案 |
| `POST /api/v1/faq/search` | 语义搜索（余弦相似度） |
| `POST /api/v1/faq/copilot` | AI 问答助手（搜索+LLM 生成推荐回复） |
| `GET /api/v1/faq/stats` | 知识库统计概览 |

**技术实现：**
- 问答对提取：Dashscope qwen3-max（LLM 智能清洗，过滤寒暄/图片/无效消息）
- 语义向量化：Dashscope text-embedding-v3（1024 维度，批量 20 条/次）
- 问题聚类：scikit-learn AgglomerativeClustering（余弦距离，阈值 0.35，average linkage）
- 集群精炼：LLM 生成标题/分类/代表性问题/最佳答案/答案质量评分
- Copilot：向量相似度 Top-K + LLM 综合推荐回复

### 3.7 权限审计（2026-03-13 新增）

- 权限体检：`GET /api/v1/admin/permission-audit`（仅超管）
- 一键修复：`POST /api/v1/admin/permission-audit/fix`
- 审计维度：数据库实际值 vs 运行时判定、默认放行/定制权限/JSON 异常、兼容逻辑隐式放行检测

### 3.8 系统管理

- 系统设置：站点名称、品牌色、功能开关（AI 评分/导出中心/审计增强）
- 租户管理：创建/停用租户、代入模式
- 数据导出：用户/练习/题库 CSV 导出
- 审计日志：操作记录追踪

## 7. 与原规划文档差异（维护提示）

- 规划文档包含 Redis、lxml 等可选项；当前落地版本未作为运行必需。
- 训练推进逻辑已按业务反馈调整为"单次回复即推进"。
- "主动/被动聊天"在导入阶段手动选择，等待上游导出结构化字段后再升级自动识别。
- FAQ 知识库为新增模块，不在原始规划内，但复用了现有的对话导入基础设施。
- 权限审计为超管运维工具，未在原始规划内。
- 后续 SaaS 模块化（功能可插拔、租户按需开关）正在规划中——详见主项目文档 12.23 节。
